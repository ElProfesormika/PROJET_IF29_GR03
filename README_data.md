# Dataset utilisateurs – Projet IF29

Source :
- Base MongoDB : database_local
- Collection : users_aggregated

Description :
- 1 ligne = 1 utilisateur Twitter
- Données agrégées à partir de 1 161 999 tweets
- Aucun label appliqué

Variables principales :
- followers_count
- friends_count
- statuses_count
- favourites_count
- verified
- nb_tweets_observed
- nb_retweets
- avg_favorite_count
- avg_retweet_count
- follower_friend_ratio
- retweet_ratio

Utilisation :
- Données prêtes pour analyse exploratoire
- Compatibles avec méthodes non supervisées et supervisées
